# Eureka Server

Componente del ecosistema Spring Cloud encargado del Service Discovery (Directorio de Servicios).

## Instrucciones de Uso
1. Debe ser el **primer servicio** en levantarse en la arquitectura.
2. Ejecutar con `mvn spring-boot:run` o a través de Docker.
3. El puerto por defecto es `8761`.
4. El panel de administración de Eureka estará disponible en `http://localhost:8761`.
